# Neo4j Vue

## 安装步骤
1. 克隆或下载该库到本地
2. 下载neo4j并安装到本地, 启动neo4j服务
3. 进入到该工程的根目录, 输入命令: npm run update 进行npm包的导入
4. 使用命令: npm run dev 启动该工程
5. 默认端口是:8081
   - neo4j账号信息:
   - url: bolt://localhost
   - userName: neo4j
   - passsword: password

### 启动服务
`npm run dev` and point your web browser to `http://localhost:8081`.
